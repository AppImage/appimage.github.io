## NORUN

NORUN allows you to run Windows applications on Linux using Wine or Proton
with optional sandboxing, virtual desktops, and isolated prefixes.

### Features
- CLI + GUI
- Wine / Proton support
- DXVK toggle
- Virtual desktop mode
- Portable AppImage

### Download

```bash
wget https://github.com/HasibAlTahsin/norun/releases/latest/download/Norun-x86_64.AppImage
chmod +x Norun-x86_64.AppImage
./Norun-x86_64.AppImage --help
